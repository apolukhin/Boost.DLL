Function import_function_alias

boost::dll::import_function_alias

Synopsis

// In header: <boost/dll/import_function.hpp>


template<typename T> 
  boost::function< T > 
  import_function_alias(const boost::filesystem::path & lib, 
                        boost::string_ref func_name, 
                        load_mode::type mode = load_mode::default_mode);
template<typename T> 
  boost::function< T > 
  import_function_alias(const boost::shared_ptr< shared_library > & lib, 
                        boost::string_ref func_name);

Description

Imports function by it alias name and returns boost::function<T> that holds the function from the loaded library and refcounts usage of the loaded shared library. Library won't get unload until all copies of returned value are not destroyed.

This call will succeed if call to boost::dll::shared_library::search_symbol(boost::string_ref ) function with the same symbol name returned true.

For non alias names use boost::dll::import_function method.

Examples: 

boost::function<int(int)> f = import_function_alias<int(int)>(
          boost::make_shared<shared_library>("test_lib.so"),
          "integer_func_alias_name"
);

boost::function<int(int)> f = import_function_alias<int(int)>("test_lib.so", "integer_func_alias_name");

Parameters:

func_name

Null-terminated name of the function to import. Can handle std::string, char*, const char*.

lib

Shared pointer to library to load function from.

mode

An mode that will be used on library load.

Returns:

boost::function<T> that holds an imported function from the loaded library and refcounts usage of the loaded shared library.

Throws:

 boost::system::system_error if symbol does not exist or if the DLL/DSO was not loaded. Overload that accepts path also throws std::bad_alloc in case of insufficient memory.
